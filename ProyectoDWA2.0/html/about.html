AboutCuando hablamos de internet, pensamos instantáneamente en la palabra conexión; y es que sería difícil encontrar una característica más adecuada para relacionar a la internet.

Cómo funciona IoT

De forma simple Internet de las Cosas trata de objetos conectados entre si por medio de la red. Estos intercambian información para facilitar o crear diversas acciones.

Los Dispositivos

Son todos aquellos que ya conocemos, como: refrigeradores, carros, lamparas, relojes, cafeteras, televisión y otros.

En estos dispositivos es importante que sean equipados con los items correctos para proporcionar la comunicación con los demás elementos. Esos límites pueden ser chips, conexión con internet, sensores, antenas entre otros.

La Red

Es el medio de comunicación y ya estamos acostumbrados a ella. Pues son tecnologías como Wi-Fi, Bluetooth y datos móviles (3G y 4G).

El sistema de control

Es necesario para que todos los datos capturados de los dispositivos a través de red sean procesados, entonces ellos son enviados para un sistema que controla cada aspecto y hace nuevas conexiones.

APIs Implementadas

API de WhatsApp

API significa Interfaz de Programación de Aplicaciones y en ella se recogen todos los requisitos, instrucciones y posibilidades para integrar WhatsApp en otras herramientas o automatizar ciertas tareas.

API de PayPal

Gracias a la API de PayPal, los clientes podrán realizar pagos desde esta plataforma en tu tienda online.

API de Pexels

La API de Pexels te da acceso a toda nuestra biblioteca de fotos y vídeos, con contenido nuevo a diario para que tus proyectos y diseños estén al día con las últimas tendencias visuales.

API de Google Maps

Es un tipo de identificación (Interfaz de Programación de Aplicaciones) para el uso de aplicaciones con Google Maps.
